# StompLab makes prototyping easy!

StompLab is a breadboard plugin module that provides a set of nice features for you to get going with your prototype!

* Bypass
* Power rails
  * GND
  * Vin (4-24 V)
  * Buffered Vin/2
* Stereo input
  * Optional buffers
* Stereo output
  * Optional buffers
