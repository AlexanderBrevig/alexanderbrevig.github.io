## About

A utility for generating markdown documentation from KiCAD libraries

## Article

A Rust CLI that extracts symbols and footprints from KiCAD libraries and generates markdown documentation tables with datasheets and footprint images. Useful if you maintain a KiCAD library. It solves a specific documentation workflow. Well-scoped.
